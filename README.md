# 학생 정보 조회 페이지

HTML, CSS, JavaScript만으로 만든 정적 학생 정보 조회 페이지입니다. 별도 서버나 빌드 도구 없이 브라우저에서 바로 실행할 수 있습니다.

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 실행됩니다. 별도 백엔드, 데이터베이스, 빌드 도구가 필요하지 않습니다.

## 실습 계정

| 구분 | ID | PW |
|---|---|---|
| 관리자 | `admin` | `2026` |
| 학생 1 | `10101` | `1234` |
| 학생 2 | `10102` | `1234` |
| 학생 3 | `10103` | `1234` |

## 구성

- `index.html`: 페이지 구조
- `style.css`: 화면 스타일
- `app.js`: 로그인 처리와 학생 정보 렌더링
- `assets/`: 학생 사진 이미지
